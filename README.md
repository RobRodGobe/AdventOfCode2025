# AdventOfCode2025
Code in C# for AoC 2025

The project runs under .NET 8, so that version of the SDK is a requirement. This is a console application that receives the day number as an argument (i.e. dotnet run 1) and returns the responses for both parts of the day. Modify the input text files as needed. 

To run all the days, use dotnet run all.

This includes the binaries used from Z3.
